Movement.js是使用JavaScript和HTML5识别基本手势的库。

使用方法

初始化

该callbacks对象可以包含以下回调：

  • movementChanged - 用户手势更改时将调用的回调。
  • positionChanged - 当用户在可见区域中改变其位置时将调用的回调。

两个回调都接受新回调/移动的单个参数。

运动清单

这是Movement.js当前可以识别的所有手势的列表：

  • STAND
  • LEFT_ARM_UP
  • RIGHT_ARM_UP
  • ARMS_UP
  • SQUAT_LEFT_ARM_UP
  • SQUAT_RIGHT_ARM_UP
  • LEFT_LEG_UP
  • SQUAT
  • EMPTY
  • RIGHT_LEG_UP

所有移动都位于命名空间中Movement.movements。

list of position

his is list of all positions which Movement.js supports currently:

  • LEFT
